# Linear algebra. Rust

[![Language](https://img.shields.io/badge/Language-Rust-black?style=flat&logo=rust)](https://www.rust-lang.org/)
[![License: MIT](https://img.shields.io/badge/License-MIT-yellow.svg)](https://opensource.org/licenses/MIT)

> A lightweight, efficient, and purely Rust library for Linear Algebra.

`linalg-rs` is a custom linear algebra library designed to provide easy-to-use data structures and algorithms for mathematical computations involving vectors and matrices.

## Features
- **Core Structures:** Support for Vectors, Matrices, Sparse matricies.
- **Arithmetic Operations:** Integrated with Rust's `std::ops` for ergonomic and intuitive syntax.
- **Linear Algebra Algorithms:**
  - Decompositions.
  - Iterative SLAE solvers.
  - Eigen problem solvers.
- **Zero-cost Abstractions:** Designed with Rust's ownership and borrowing models in mind to ensure high performance.
